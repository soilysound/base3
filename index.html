Font sizes typeography.css

Provides a list of font size css variables that either scale between the mobile and desktop views or ar fixed at all views, aswell as line-height variables that corrospond to each font size.

Helper css classes are also available that bundle in the correct line height for that font size value, as well as a .font-size-any helper class that lets you plug in two values and scale between them.

Helper css classes are also available, as well as a .gap-any helper class that lets you plug in two values and scale between them.
